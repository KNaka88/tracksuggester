#Koji Nakagawa

#### 2nd Week Independent Project for Epicodus, 01.13.2017

#### By Koji Nakagawa

## Description
* This website is second independent project for Epicodus "Intro to Programming" class.
* This website offers suggestions for users to choose which Epicodus course track is better for them.


## Setup/Installation Requirements

* Clone this repository.
* Open the index.html by your web browser.
